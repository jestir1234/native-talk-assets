🚶‍♀️ 核心动词一：行く (iku) - 去

行く (いく / iku) 是一个非常重要的动词，表示“去”、“前往”。当表示去某个地点时，通常使用助词 に (ni) 来指明目的地。

行く (いく)
去 / 前往

例句：

わたしは がっこう に いきます。
我去学校。

わたし (watashi): 我
がっこう (gakkou): 学校
に (ni): 助词，表示目的地
いきます (ikimasu): 行く 的礼貌形（现在/将来时）

えき に いきます。
去车站。

えき (eki): 车站

🍚 核心动词二：食べる (taberu) - 吃

食べる (たべる / taberu) 是表示“吃”的动词。当表示吃什么东西时，通常使用助词 を (o) 来指明宾语。

食べる (たべる)
吃

例句：

パン を たべます。
吃面包。

パン (pan): 面包
を (o): 助词，表示宾语
たべます (tabemasu): 食べる 的礼貌形

ごはん を たべます。
吃饭。

ごはん (gohan): 饭

✨ 核心动词三：する (suru) - 做

する (suru) 是一个万能动词，表示“做”、“进行”。它可以和名词结合，形成新的动词短语，比如“学习”、“购物”等。

する (する)
做 / 进行

例句：

べんきょう を します。
学习。

べんきょう (benkyou): 学习 (名词)
します (shimasu): する 的礼貌形

かいもの を します。
购物。

かいもの (kaimono): 购物 (名词)

👋 核心动词四：来る (kuru) - 来

来る (くる / kuru) 表示“来”、“来到”。和 “行く” 类似，当表示从某处“来”时，也常与助词 に (ni) 结合。

来る (くる)
来 / 来到

例句：

ともだち が きます。
朋友来了。

ともだち (tomodachi): 朋友
が (ga): 助词，表示主语
きます (kimasu): 来る 的礼貌形

いえ に きます。
来家里。

いえ (ie): 家

✏️ 动词活用基础：现在/将来肯定形 (ます 形)

在日常交流中，我们通常使用动词的 ます 形 (masu-kei)，它是一种礼貌的表达方式，可以表示现在或将来的动作。

基本规则（本章动词）：

  • 行く (iku) → 行きます (ikimasu)
  • 食べる (taberu) → 食べます (tabemasu)
  • する (suru) → します (shimasu)
  • 来る (kuru) → 来ます (kimasu)

🙅‍♀️ 动词活用基础：现在/将来否定形 (ません 形)

要表达否定，只需将 ます 替换为 ません (masen)。这表示“不…”或“不曾…”。

基本规则：

  • 行きます (ikimasu) → 行きません (ikimasen)
  • 食べます (tabemasu) → 食べません (tabemasen)
  • します (shimasu) → しません (shimasen)
  • 来ます (kimasu) → 来ません (kimasen)

 不吃

 不做

 不来

否定形例句：

パン を たべません。
不吃面包。
きょう は がっこう に いきません。
今天不去学校。

きょう (kyou): 今天

📚 本章总结

恭喜你！本章你学习了日语的四个核心动词以及它们的基本活用形式：

  • 核心动词：
    • 行く (いく)
      去
    • 食べる (たべる)
      吃
    • する (する)
      做
    • 来る (くる)
      来
  • 动词活用：
    • ます 形 (礼貌形 / 现在或将来肯定)：
      例：行きます (ikimasu) - 去 / 食べます (tabemasu) - 吃
    • ません 形 (礼貌形 / 现在或将来否定)：
      例：行きません (ikimasen) - 不去 / 食べません (tabemasen) - 不吃
  • 基本句型：
    • [主语]は [地点]に [动词ます形/ません形]。 (去/来某个地方)
    • [名词]を [动词ます形/ません形]。 (吃/做某物)
