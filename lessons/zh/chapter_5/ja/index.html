Chapter 5: 家族と人間関係

🎉 こんにちは！第5章へようこそ！

この章では、中国語での家族メンバーの呼び方、そして「私の〜」といった所有を示す表現を学びます。中国語の家族表現は日本語と似ている部分もあれば、少しユニークな部分もあります。一緒に楽しく学んでいきましょう！

👨‍👩‍👧‍👦 セクション1: 家族メンバーの紹介

まずは、一番基本的な家族メンバーの呼び方から見ていきましょう。

1. 両親

爸爸 bàba お父さん
妈妈 māma お母さん

2. 兄弟姉妹

中国語では、日本語のように「兄」「姉」「弟」「妹」と区別します。特に、年上か年下かを明確にすることが重要です。

哥哥 gēge お兄さん
姐姐 jiějie お姉さん
弟弟 dìdi 弟
妹妹 mèimei 妹
这是我的爸爸。 これは私のお父さんです。
她是我姐姐。 彼女は私の姉です。

💡 セクション2: 所有代名詞「的 (de)」

「〜の」や所有を示すときには、助詞の「的 (de)」を使います。日本語の「の」に非常に似ています。

基本的な形: [所有者] + 的 (de) + [名詞]

我的 wǒde 私の
你的 nǐde あなたの
他的 tāde 彼の
她的 tāde 彼女の
他们的 tāmen de 彼らの / 彼女らの

ポイント： 家族メンバーなど親しい関係性の名詞の前では、「的 (de)」を省略することがよくあります。例: 我的妈妈 → 我妈妈

我的书。 私の本。
我妈妈。 (より自然な表現) 私のお母さん。

🏡 セクション3: 「これは私の〜です」

誰かを紹介するときによく使う表現です。

[これは/彼は/彼女は] + 私の + [家族メンバー]

中国語では：这是我的 [家族メンバー]。 / 他是我的 [家族メンバー]。 / 她是我的 [家族メンバー]。

※「これ/これら」は「这 (zhè)」を使いますが、人の紹介では「彼/彼女」を表す「他 (tā) / 她 (tā)」もよく使います。

这是我弟弟。 これは私の弟です。
他是我哥哥。 彼はお兄さんです。
她是我妹妹。 彼女は私の妹です。

👨‍👩‍👧‍👦 セクション4: その他の家族語彙と関係

基本的な家族メンバー以外にも、覚えておくと便利な語彙があります。

1. その他の親族

爷爷 yéye （父方の）おじいさん
奶奶 nǎinai （父方の）おばあさん
外公 wàigōng （母方の）おじいさん
外婆 wàipó （母方の）おばあさん
儿子 érzi 息子
女儿 nǚ'ér 娘
文化メモ：父方と母方のおじいちゃん・おばあちゃん👵👴

中国語では、父方と母方のおじいちゃん・おばあちゃんの呼び方が異なります。これは家族の系譜を重視する文化が反映されています。日本語にはない概念なので、少し混乱するかもしれませんが、とても重要です！

  • 父方: 爷爷 (yéye) / 奶奶 (nǎinai)
  • 母方: 外公 (wàigōng) / 外婆 (wàipó)

2. 家族全般の用語

家庭 jiātíng 家庭、家
家人 jiārén 家族（メンバー）

💬 セクション5: 家族について話す

家族について尋ねたり答えたりする基本的なフレーズです。

1. 家族の人数を尋ねる

你家有几口人？ あなたの家族は何人ですか？

口 (kǒu) は、家族の人数を数えるときに使う量詞（助数詞）です。日本語の「人」に相当します。

我家有四口人。 私の家族は4人です。
我家有爸爸、妈妈、哥哥和我。 私の家は父、母、兄、そして私です。

2. 兄弟姉妹の有無を尋ねる

你有没有兄弟姐妹？ 兄弟姉妹はいますか？

兄弟姐妹 (xiōngdì jiěmèi) は「兄弟姉妹」という意味です。「有没有 (yǒu méiyǒu)」は「あるかないか」つまり「〜ですか？」と尋ねる時に使います。

我有两个哥哥。 私には兄が2人います。
我没有兄弟姐妹。 私には兄弟姉妹がいません。
文化メモ：一人っ子政策の影響👪

中国ではかつて一人っ子政策があったため、兄弟姉妹がいない人も多くいます。このため「兄弟姐妹」の有無を尋ねる質問は一般的です。

問題2: 以下の日本語を中国語に翻訳してみましょう。（音声を聞いて発音も練習！）

1. これは私の妹です。

→ 这是我妹妹。

2. 彼らは私の家族です。

→ 我有一个弟弟。
