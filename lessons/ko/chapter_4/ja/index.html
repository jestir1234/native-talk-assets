Chapter 4: 色と描写

🎨 色を覚えよう (색을 배워요)

韓国語で「色」は 색깔 (セッカル) またはシンプルに 색 (セク) と言います。基本的な色の名前を学びましょう！

빨간색 (パルガンセク)
赤色
파란색 (パランセク)
青色
노란색 (ノランセク)
黄色
초록색 (チョロクセク)
緑色
검은색 (コムンセク)
黒色
흰색 (ヒンセク)
白色

💡 「〜色です」と表現しよう

「〜色です」と言うには、色に「입니다 (イムニダ)」または「이에요/예요 (イエヨ/イエヨ)」を付けます。

例：

빨간색입니다. (パルガンセギムニダ)
赤色です。（丁寧な表現）
파란색이에요. (パランセギエヨ)
青色です。（日常的な表現）
「黄色です」を韓国語で言ってみましょう。

📏 形容詞で描写しよう (형용사로 묘사해요)

物の大きさ、長さ、見た目などを表現する形容詞を学びましょう。

크다 / 작다 (大きい / 小さい)

크다 (クダ)
大きい
작다 (チャクタ)
小さい

💡 名詞を修飾する形： 語幹に「-은 / -ㄴ」を付けます。語幹の最後にパッチムがない場合は「-ㄴ」、ある場合は「-은」を付けます。

  • 크다 → 큰 (大きい) (クン)
  • 작다 → 작은 (小さい) (チャグン)

例：

큰 집 (クン チプ)
大きい家
작은 가방 (チャグン カバン)
小さいカバン

길다 / 짧다 (長い / 短い)

길다 (キルダ)
長い
짧다 (チャルプタ)
短い

💡 名詞を修飾する形：

  • 길다 → 긴 (長い) (キン)
  • 짧다 → 짧은 (短い) (チャルブン)

例：

긴 연필 (キン ヨンピル)
長い鉛筆
짧은 치마 (チャルブン チマ)
短いスカート

예쁘다 / 못생기다 (綺麗だ / 醜い、ブサイクだ)

예쁘다 (イェップダ)
綺麗だ、可愛い
못생기다 (モッセンギダ)
醜い、ブサイクだ

💡 名詞を修飾する形：

  • 예쁘다 → 예쁜 (綺麗な) (イェップン)
  • 못생기다 → 못생긴 (醜い) (モッセンギン)

例：

예쁜 꽃 (イェップン コッ)
綺麗な花
못생긴 개구리 (モッセンギン ケグリ)
醜いカエル
文化ノート： 「못생기다 (モッセンギダ)」は直訳すると「醜い」となりますが、韓国語では友人同士で冗談を言い合う時や、キャラクターなどに対して、日本で言う「ブサイク」のような軽いニュアンスで使われることもあります。ただし、人に対して使う際は相手に配慮が必要です。

좋다 / 나쁘다 (良い / 悪い)

좋다 (チョッタ)
良い
나쁘다 (ナップダ)
悪い

💡 名詞を修飾する形：

  • 좋다 → 좋은 (良い) (チョウン)
  • 나쁘다 → 나쁜 (悪い) (ナップン)

例：

좋은 생각 (チョウン センガク)
良い考え
나쁜 습관 (ナップン スプクァン)
悪い習慣

📝 描写動詞を使った文 (묘사 동사를 사용한 문장)

学んだ色と形容詞を組み合わせて、もっと豊かな表現をしてみましょう。

「〜は〜です」の形： 名詞 + 이/가 + 形容詞の語幹 + -아요/어요

例：

빨간색 가방이 예뻐요. (パルガンセク カバンイ イェッポヨ)
赤いカバンが綺麗です。
파란색 하늘이 좋아요. (パランセク ハヌルリ チョアヨ)
青い空が良いです。
검은색 신발이 작아요. (コムンセク シンバルリ チャガヨ)
黒い靴が小さいです。

🌟 今日のまとめ (오늘의 요약)

Chapter 4では、色と物を描写する言葉をたくさん学びましたね！

  • 基本的な色： 빨간색 (赤), 파란색 (青), 노란색 (黄), 초록색 (緑), 검은색 (黒), 흰색 (白)
  • 「〜色です」は ~색이에요/입니다 を使います。
  • 物の形や性質を表す形容詞：
    • 크다 (大きい) / 작다 (小さい)
    • 길다 (長い) / 짧다 (短い)
    • 예쁘다 (綺麗だ) / 못생기다 (醜い)
    • 좋다 (良い) / 나쁘다 (悪い)
  • 名詞を修飾する時は、形容詞の語幹に -은/ㄴ を付けます。（例：큰, 작은, 예쁜）
  • 「〜は〜です」と描写する文は、「名詞 + 이/가 + 形容詞-아요/어요」の形を使います。

これらの言葉を使って、身の回りのものを描写する練習をしてみてくださいね！お疲れ様でした！
