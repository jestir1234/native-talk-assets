Chapter 9: Weather and Seasons (天気と季節)

はじめに (Introduction) 👋

皆さん、こんにちは！今日のレッスンでは、韓国の「天気」と「季節」について学びます。旅行中に現地の天候を尋ねたり、季節の話題で会話を盛り上げたりするのに役立ちますよ！さあ、天気の話から始めましょう！

1. 날씨 (天気) ☀️

まずは、天気に関する基本的な言葉を覚えましょう。韓国語で「天気」は 날씨 と言います。

  • 날씨 (ナルッシ / nal-ssi) : 天気

今日の天気は？

「今日の天気はどうですか？」と尋ねる時は、こう言います。

오늘 날씨 어때요? (オヌル ナルッシ オッテヨ？ / oneul nalssi eottaeyo?)

→ 今日の天気はどうですか？

2. 맑다 / 흐리다 (晴れ / 曇り) ☁️☀️

晴れや曇りを表す言葉です。形容詞なので、文末で「〜です」とするには -아요/-어요 をつけます。

  • 맑다 (マルクタ / mak-da) : 晴れる、澄む
    → 丁寧語：맑아요 (マルガヨ / malg-ayo)
  • 흐리다 (フリダ / heu-ri-da) : 曇る
    → 丁寧語：흐려요 (フレヨ / heu-ryeo-yo)

날씨가 맑아요. (ナルッシガ マルガヨ / nalssiga malgayo.)

→ 天気が晴れています。

하늘이 흐려요. (ハヌリ フレヨ / haneuri heuryeo-yo.)

→ 空が曇っています。

3. 덥다 / 춥다 (暑い / 寒い) 🥵🥶

気温に関する形容詞です。これらも -아요/-어요 をつけて丁寧語にします。

  • 덥다 (トプタ / deop-da) : 暑い
    → 丁寧語：더워요 (トウォヨ / deo-wo-yo)
  • 춥다 (チュプタ / chup-da) : 寒い
    → 丁寧語：추워요 (チュウォヨ / chu-wo-yo)

오늘은 더워요. (オヌルン トウォヨ / oneureun deo-wo-yo.)

→ 今日は暑いです。

날씨가 너무 추워요. (ナルッシガ ノム チュウォヨ / nalssiga neomu chu-wo-yo.)

→ 天気がとても寒いです。

4. 비가 오다 / 눈이 오다 (雨が降る / 雪が降る) ☔️❄️

雨や雪が降ることを表す動詞表現です。

  • 비 (ピ / bi) : 雨
  • 눈 (ヌン / nun) : 雪
  • 오다 (オダ / o-da) : 来る、降る
    → 丁寧語：와요 (ワヨ / wa-yo)

「〜が降る」と言う時は、主格助詞 가/이 を使って 비가 오다 (雨が降る)、눈이 오다 (雪が降る) と表現します。

비가 와요. (ピガ ワヨ / biga wa-yo.)

→ 雨が降っています。

눈이 와요. (ヌニ ワヨ / nuni wa-yo.)

→ 雪が降っています。

5. 봄, 여름, 가을, 겨울 (春、夏、秋、冬) 🌸🏖️🍂☃️

韓国には美しい四季があります。それぞれの季節の名前を覚えましょう。

  • 봄 (ポム / bom) : 春 (3月〜5月)
  • 여름 (ヨルム / yeo-reum) : 夏 (6月〜8月)
  • 가을 (カウル / ga-eul) : 秋 (9月〜11月)
  • 겨울 (キョウル / gyeo-ul) : 冬 (12月〜2月)

저는 봄을 좋아해요. (チョヌン ポムル チョアヘヨ / jeoneun bomeul joahaeyo.)

→ 私は春が好きです。

한국의 겨울은 추워요. (ハングゲ キョウルン チュウォヨ / han-gugui gyeo-ureun chuwoyo.)

→ 韓国の冬は寒いです。

💡文化ノート：韓国の四季

韓国は日本と同じく四季がはっきりしており、それぞれの季節に美しい景色や行事があります。

  • 春 (봄): 桜やレンギョウ、ツツジが咲き乱れ、お花見が楽しめます。
  • 夏 (여름): 非常に暑く、湿度も高いですが、海水浴や渓谷での休暇が人気です。
  • 秋 (가을): 紅葉が美しく、最も過ごしやすい季節と言われます。澄んだ青空が特徴です。
  • 冬 (겨울): 非常に寒く、雪も多く降ります。スキーやスノーボードを楽しむ人も多いです。

6. Temperature Expressions (気温の表現) 🌡️

気温を尋ねたり答えたりする方法です。

  • 몇 도예요? (ミョットエヨ？ / myeot do-yeyo?) : 何度ですか？
  • ~도예요. (〜ドエヨ / ~do-yeyo.) : 〜度です。
    (例: 20도예요. (イシップ ドエヨ / iship do-yeyo.) → 20度です。)

A: 오늘 몇 도예요? (オヌル ミョットエヨ？ / oneul myeot do-yeyo?)

→ 今日は何度ですか？

B: 지금 10도예요. (チグム シットエヨ / jigeum ship do-yeyo.)

→ 今10度です。

※ 韓国語では「零下」は 영하 (ヨンハ / yeong-ha) 、「零上」は 영상 (ヨンサン / yeong-sang) と言います。

영하 5도예요. (ヨンハ オドエヨ / yeongha o do-yeyo.)

→ 零下5度です。

7. Seasonal Activities and Clothing (季節の活動と服装) 👕🧣

各季節によくする活動や着るものについて話す時に役立つ表現です。

活動 (활동 / hwal-dong)

  • 꽃구경하다 (コックギョンハダ / kkot-gu-gyeong-hada) : お花見をする (春)
  • 수영하다 (スヨンハダ / su-yeong-hada) : 泳ぐ (夏)
  • 단풍 구경하다 (タンプン グギョンハダ / dan-pung gu-gyeong-hada) : 紅葉狩りをする (秋)
  • 스키 타다 (スキ タダ / seu-ki ta-da) : スキーをする (冬)

服装 (옷 / ot)

  • 반팔 (パンパル / ban-pal) : 半袖 (夏)
  • 긴팔 (キンパル / gin-pal) : 長袖 (春、秋)
  • 외투 (ウェトゥ / oe-tu) : コート (冬)
  • 장갑 (チャンガプ / jang-gap) : 手袋 (冬)
  • 목도리 (モクドリ / mok-do-ri) : マフラー (冬)

여름에는 수영해요. (ヨルメヌン スヨンヘヨ / yeoreumeneun su-yeonghaeyo.)

→ 夏には泳ぎます。

겨울에는 외투를 입어요. (キョウレヌン ウェトゥルル イボヨ / gyeouleneun oetoreul ibeoyo.)

→ 冬にはコートを着ます。

練習してみましょう！ (Let's Practice!) ✍️

学んだ表現を使って、以下の日本語の文を韓国語にしてみましょう。

1. 今日の天気は晴れです。

→ 오늘 날씨는 맑아요.

2. 雨が降っています。

→ 비가 와요.

3. 今、とても寒いです。

→ 지금 너무 추워요.

4. 韓国は秋が美しいです。

→ 한국은 가을이 예뻐요.

5. 明日は何度ですか？

→ 내일 몇 도예요?

今日のまとめ ✨

Chapter 9では、天気と季節に関する重要な韓国語表現を学びました。

  • 天気の表現: 날씨 (天気), 맑다 (晴れる), 흐리다 (曇る), 덥다 (暑い), 춥다 (寒い), 비가 오다 (雨が降る), 눈이 오다 (雪が降る)。
  • 四季: 봄 (春), 여름 (夏), 가을 (秋), 겨울 (冬)。
  • 気温: 몇 도예요? (何度ですか？), ~도예요. (〜度です)。
  • 関連語彙: 季節ごとの活動や服装。

これで、韓国の方と天候や季節の話題で会話を楽しめるようになりましたね！日常生活でも積極的に使ってみてください。
